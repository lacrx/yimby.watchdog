# Technical Assistance and Enforcement Letters Dashboard

The Housing Accountability Unit issues letters to local agencies regarding the review of local ordinances, provision of technical assistance, and enforcement of various state housing laws. HCD makes these letters and actions available to the public via the Technical Assistance and Enforcement Letters Dashboard below. The dashboard contains all letters issued and has filters that can be used to search by jurisdiction, date, subject matter, and statutory reference.

## Dashboard User Guide

### To view/download a letter:

Use the “HCD Letter” field. Any underlined text in that field represents a link that you can click on to view the corresponding letter in PDF format.

For letters that are not yet accessible through a link, the field will state to “contact HCD for a copy” of the letter. The email to contact is [HAUportal@hcd.ca.gov](mailto:HAUportal@hcd.ca.gov).

### Data Dictionary:

Each letter sent by the HAU is represented in a single row on the dashboard and is characterized according to 9 descriptive fields:

- **Case Number** – The unique ID assigned to the case that resulted in the HCD Letter when the case first entered the HAU or ADU Portal.
- **Letter Date** – The date the letter was sent by HCD.
- **Jurisdiction** – The jurisdiction that received the letter.
- **Housing Law** – The type of State housing law relevant to the case for which the letter was sent (Ex: Density Bonus Law, Housing Element Law, etc.).
- **Letter Type** – The type of letter provided by HCD (Ex: Notice of Violation, Technical Assistance Letter, Ordinance Findings Letter)
- **HCD Letter** – The method for accessing a copy of the letter (see above).
- **Task** – Unique ID of the letter (mostly for HCD internal use).
- **Statutory Reference** (filter only) – The sections of statute relevant to the letter.
- **Keyword** (filter only) – References to more specific types of housing issues that are mentioned in the letter or relevant to the case for which the letter was sent (Ex: “By-right supportive housing”, “Maximum allowable density”, “Parking”).

### Filtering and sorting:

Each of the six boxes at the top of the dashboard represent criteria that can be used to filter which letters are displayed in the dashboard. All filters can be active at once and can be used to create a set of conditions that must be met for a letter to be displayed.

The filters for Jurisdiction, Housing Law, Letter Type, Statutory Reference, and Keyword contain search bars that allow a user to search for specific options within that filter and also allow for the selection of multiple options from the same filter for inclusion in the display. The options listed under each filter represent all the values recorded for that field across all the HCD letters in the dashboard. If a search for certain criteria is returning no results, that either means the search should be refined or rephrased, or that no HCD letters in the dashboard are related to the specific search criteria.

In addition to filtering which letters are displayed, users may also sort the letters displayed on the dashboard according to the values in a given field by clicking each column header. For example, clicking on the Jurisdiction column header in the dashboard would sort the displayed letters in alphabetical order by jurisdiction.

### Notes on Letters not Included:

Letters related to the Surplus Land Act (SLA) or Affordable Housing Preservation Law (AHPL) are not included in this dashboard. They are publicly available on the [Surplus Land Act web page](https://www.hcd.ca.gov/planning-and-research/public-lands/surplus-land "Surplus Local Land for Affordable Housing") and [Affordable Housing Preservation Law web page](https://www.hcd.ca.gov/planning-and-research/preserving-existing-affordable-housing "Preserving Existing Affordable Housing") respectively.
